Join The Server

For those who have been whitelisted, the server is accessable using the ip in my discord under the

#server-info

channel.


In the same channel you can also find the current server version and other information.

Clients?

The server is avaliable to join on Vanilla Java/Bedrock Edition, or vanilla compatible clients like Optifine.


However the server has certain custom Vanilla+ features, like Voice Chat. Thus I recommend using a modpack/client that includes these features, if you don't know what to download I host my own modpacks/clients at MinecraftCustomClient/Downloads.

Resourcepacks?

The server does apply a custom resourcepack when you join to include ex. custom-item-textures. Information for the resourcepack can be found at sbamboo.github.io/websa?id=lt



If you are having issues with the resourcepack, you can disable it for the server and download it yourself from https://github.com/sbamboo/serverpacks.

Rules

In general the rules can be summarized as to not ruin the experience for anyone else. Use common sense!


Specifics
  • No cheating, or doings that in otherways grant an unfair advantage.
  • Griefing is never allowed, nor is destruction of other peoples things in general,
    if someone has built things where they aren't suppose to and won't remove it, contact an admin.
  • Offensive actions/messages towards someone is never allowed.
    This includes chat-messages, custom-sounds, builds and ingame-texts like signs.
    This should be a safe community for anyone!
  • The server in general has no building rules, however each base/town/city/region can define their own rules.
    Note, the server's general rules always apply!
Is X mod allowed?
  • Using a minimap? Disable cave-mode so you only see the world from the top.
  • Bot-automations like Baritone might be allowed, but generally not, ask server-owner first.
  • Freecam is allowed, but you may not use it as a cheat.
