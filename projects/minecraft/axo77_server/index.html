About

Welcome to the about page for @TheAxolot77's Minecraft Server!

The server is made to be a place for me and my friends to play together, but also my goto minecraft world. And builds on the concept of having one, sometimes abit too large, world where you can collect, your builds, experiences and memories. Where other servers have seasons and resets, we instead run away to a new place and start anew, allowing us to keep all our previous shenanigans while getting something new and fresh.

While this is not the playstyle everyone enjoys, if you got the urge to run away it might be for you!

The server features some Vanilla+ mods like VoiceChat and allowing bedrock players, while still allowing vanilla clients to join, this includes even custom made mods!
